## Scaling Plan

1. **Weeks 1–2:** Run all campaigns, gather data, pause low-performers
2. **Weeks 3–4:** Optimize bids based on CPA data, add negative keywords from search terms
3. **Month 2:** Increase budget on winning campaigns, test new ad copy
4. **Month 3:** Switch to Target CPA bidding if 30+ conversions achieved
5. **Ongoing:** Shift budget toward county campaigns with lowest CPA
