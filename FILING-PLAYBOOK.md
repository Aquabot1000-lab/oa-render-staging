## THE SYSTEM: How We File at Scale

### Universal Process (Every Client, Every County)
1. **PRE-SEASON (NOW → April 15)**
   - ✅ Form 50-162 signed (agent authorization)
   - ☐ Register agent accounts on every county portal we have clients in
   - ☐ Download/verify county parcel data
   - ☐ Pre-build evidence packets (comp analysis + E&U data)
   - ☐ Submit 50-162 to each CAD (some accept online, some need mail/email)

2. **NOTICE DROPS (mid-April)**
   - Client receives Notice of Appraised Value with Owner ID + PIN + Property ID
   - We NEED these credentials to file online (most counties require them)
   - **ACTION:** Email/text every signed client immediately: "Forward your notice or give us your Owner ID + PIN"

3. **FILE PROTEST (within 30 days of notice, or by May 15)**
   - File via county online portal (preferred) or mail Form 50-132
   - Attach Form 50-162 (agent authorization) if not already on file
   - Upload evidence packet

4. **HEARING (May-August)**
   - Informal first (phone/Zoom/online with appraiser)
   - If not resolved → ARB formal hearing
   - We attend on client's behalf (authorized by 50-162)

## FALLBACK: MAIL FILING

If any online portal fails or blocks us:
1. **Form 50-132** (Notice of Protest) — print, fill, mail
2. **Form 50-162** (Agent Authorization) — attach copy
3. **Evidence packet** — print and mail with protest
4. **Send certified mail** with return receipt (proof of filing)
5. **Addresses:** Each CAD has a mailing address on their website

**Pro tip:** Always file online AND mail a backup copy for critical cases.
