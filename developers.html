Google Ads API Usage

OverAssessed uses the Google Ads API to manage advertising campaigns for our property tax protest services. This page describes our API usage in compliance with Google's developer token policies.

About OverAssessed

OverAssessed, LLC is a property tax protest company based in San Antonio, Texas. We help homeowners across Texas, Georgia, Washington, Arizona, and Colorado reduce their property taxes through professional assessment appeals.

Website: overassessed.ai

How We Use the Google Ads API

We use the Google Ads API exclusively for internal campaign management across our own advertising accounts. We do not resell API access or operate as a third-party SaaS platform.

Managed Accounts:

  • OverAssessed — Property tax protest lead generation campaigns across multiple states
  • Worthey Aquatics — Pool construction and service company advertising in San Antonio, TX

Campaign Types:

  • Lead generation campaigns (Search, Display)
  • Traffic campaigns for landing pages
  • Local Services Ads management
  • Performance monitoring and reporting

API Usage:

  • Automated campaign performance reporting
  • Budget monitoring and optimization
  • Keyword and ad group management
  • Conversion tracking and attribution

Compliance

We comply with all Google Ads API Terms and Conditions, including:

  • API access is used only for managing our own advertising accounts
  • We do not provide API access to third parties
  • All campaigns comply with Google Ads advertising policies
  • User data is handled in accordance with our Privacy Policy
